# Referencias de Bordy 3D (EXP-2b)

David compartió 3 imágenes de referencia por chat (2026-07-17) al autorizar la ejecución de
`PLAN-BORDY-3D.md`. Los archivos originales aún no están en esta carpeta — si se quieren
conservar, arrastrarlos aquí como `ref-1-voxel.png`, `ref-2-wireframe.png`, `ref-3-kvrc.png`.
Mientras tanto, esta es la descripción de trabajo que usó Fable:

1. **Robot voxel gris** (estilo Minecraft/juguete): cuerpo por bloques, ojos rectangulares
   cian brillantes, panel de pecho naranja, antena con bola naranja. → Se toma: lectura de
   juguete, ojos luminosos como foco de la cara, un solo acento de color en el pecho.

2. **Render gris de modelado (wireframe, estilo Maya)** — ⭐ la favorita de David: cabeza
   esférica pequeña sobre un torso esférico ENORME, brazos de tubo segmentado (bendy) con
   articulaciones de bola, nada humanoide. Indicación explícita: *"me gusta que tenga
   dimensiones no uniformes, no luce como humano; no uses exactamente esas proporciones pero
   puedes jugar con las dimensiones"*. → Se toma: masas esféricas desproporcionadas entre sí y
   brazos de segmentos tipo manguera; NO se copian las proporciones literales.

3. **Robot naranja articulado (estilo K-VRC de Love, Death & Robots)**: cabeza grande con cara
   de pantalla oscura y ojos redondos blancos brillantes, cuerpo diminuto, aletas como orejas.
   → Se toma: la cara como panel de vidrio oscuro donde SOLO viven los ojos (sin boca), cabeza
   dominante sobre cuerpo pequeño.

Síntesis aplicada en `../bordy-3d-explore.html`: cabeza-globo esférica grande (guiño a la
geografía de Frontle) con cara de vidrio oscuro y ojos luminosos, cuerpo cápsula pequeño,
brazos bendy segmentados, antena-LED semáforo, paleta Violeta Prisma + oro Celo.
